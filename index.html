SIDE PROJECTS

Besides my blog, here you can find a curated list of some of my interesting and public side projects

UnrealROX
Description

The 3D Perception Lab at the University of Alicante is a group of researchers interested in the intersection of machine learning and computer vision. Our research mission focuses on various aspects of perception often related with mobile robotics in which we exploit 3D data as the main source of information.

Role: VR Developer

During my time at 3DPL I worked on UnrealROX: An eXtremely Photorealistic Virtual Reality Environment for Robotics Simulations and Synthetic Data Generation.

Experimental Arcade Vehicle
Description

Open source experimental plugin to demonstrate vehicle physics concepts using Unreal Engine 4. Features: suspension models, substepped physics, drag coefficients, variable friction, arcade air and ground controllability, based movement.

Role: Physics Programmer

I worked alongside Daniel Batten (MonsOlympus) to ensure a feature-complete open-source arcade vehicle sample plugin.
Mass Documentation
Description

Our WIP understanding of Unreal Engine 5's experimental Mass plugin (UE5 Entity Component System) with a small sample project.

Role: Documentation and Programming

I work alongside Karl (Megafunk) to provide a full fledged documentation and code samples about the Mass framework.
